▸ How to prepare your Perdix 2
On the computer: press LEFT until Bluetooth is highlighted, then press RIGHT to select.
Keep the Perdix within 30 cm of your laptop during download.
⚠️ Chrome or Edge only — Web Bluetooth is not supported in Firefox or Safari.
⚠ Passkey pairing required
The EON Steel and EON Core use BLE Secure Pairing with a numeric passkey shown on the dive computer screen. Web Bluetooth cannot initiate this passkey exchange — it is a browser-level security constraint, not a hardware limitation.
1Export via USB using Suunto DM5 (Win/Mac) or SuuntoLink
2In DM5: File → Export → choose SML or UDDF
3Drop the file into the importer below
Alternatively: Subsurface (free desktop app) handles passkey pairing on desktop and can export UDDF.
Run python suunto_bridge.py --serve --user yourname on your laptop. The bridge handles the passkey pairing that browsers cannot do.
▸ Install bridge (one-time)
1. Install dctool
  macOS: brew install libdivecomputer
  Linux: sudo apt install libdivecomputer-utils
  Windows: libdivecomputer.org/builds/stable

2. Install Python deps
  pip install requests flask lxml

3. Start bridge
  python suunto_bridge.py --serve --user kalle

4. First pairing
  EON Steel: Settings → Connectivity → Airplane Mode: Off
  Enter passkey from dive computer screen when OS dialog appears.
